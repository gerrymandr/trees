132 """
133 pointer_from_objref(x)
134
135 Get the memory address of a Julia object as a `Ptr`. The existence of the resulting `Ptr`
136 will not protect the object from garbage collection, so you must ensure that the object
137 remains referenced for the whole time that the `Ptr` will be used.
138

141

143 """
